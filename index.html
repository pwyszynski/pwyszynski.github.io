About me

Dedicated web apps developer centered around .NET and MS, but not afraid of other technologies and learning new frameworks/languages.

I have started my career with an internship focused on testing web apps with Selenium testing using Java, then moved to a full-time employee at Sii, where I got hooked with C# - at that time writing hardware validation tests. Over years, I have moved between different projects dedicated to HW validation, working with many wonderful people. However, I wanted to learn something more - web development. I focused my learning on ASP.Net and got my first job as a web developer.

Well, not strictly a web developer - a Microsoft Dynamics CRM developer at Nordea. This included working with .NET, JavaScript/TypeScript and CSS, but it was not "pure web development". Luckily, for the same project, I got to work on a separate application that was written in .NET and Angular. This has massively boosted my web dev skills.

Some time later I got an opportunity to work at Intel, developing web applications related to gaming - and as an avid gamer, I did not hesitate. Since then, I have been working on multiple of applications used by many different teams at Intel - including testing teams, marketing teams and others.
